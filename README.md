# py-tgbot-klua
Krau的telegram bot —Klua

## 功能
- 涩图
- 河南怎么你了？
- 中二发言
- 原神怎么你了
- 早安!
- 晚安!

## 使用
1. `git clone https://github.com/krau/py-tgbot-klua`
2. pip install -r requirements.txt
3. edit config.yml
4. python3 bot.py
